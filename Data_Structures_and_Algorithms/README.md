# Module: Data Structures and Algorithms

This directory contains the mandatory hands-on exercises for the Cognizant Digital Nurture 5.0 DSA track.

##  Exercises Implemented

### 1. Exercise 2: E-commerce Platform Search Function
* **Location:** `/EcommerceSearch/ProductSearch.java`
* **Concept:** Implemented an efficient search algorithm (Binary Search) to locate product IDs within an array, demonstrating time complexity optimization ($O(\log n)$) compared to linear search ($O(n)$).

### 2. Exercise 7: Financial Forecasting
* **Location:** `/FinancialForecasting/ForecastEngine.java`
* **Concept:** Utilized recursion to model financial growth, demonstrating an understanding of how recursive problem-solving can be applied to business scenarios.
